## Logarithmic units 

| Unit           | Name                            |
|----------------|---------------------------------|
| `dB`       |        Decibel |
| `B`        |         Bel |
| `Np`       |        Neper |
| `cNp`      |       Centineper |

### Log units related to reference levels
| Unit           | Reference level                            |
|----------------|---------------------------------|
| `dBHz`       |         1Hz |
| `dBm`          |         1mW |
| `dBV`          |         1V |
| `dBu`          |         sqrt(0.6)V |
| `dBμV`        |         1μV |
| `dBSPL`      |         20μPa |
| `dBFS`        |         RootPowerRatio(1) |
| `dBΩ`          |         1Ω |
| `dBS`          |         1S |
